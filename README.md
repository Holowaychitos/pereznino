## Build Setup

This is the repo for the PerezNiño Admin UI. This is a Vuejs App, with a custom flowchart interpreter.

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
